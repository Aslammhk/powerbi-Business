## ✅ Features

### Core System
- 🌳 Tree-based referral network
- 🤖 Auto-browse 25 channels per member
- 🔒 Admin channel lock (tamper-proof)
- 📊 Upward tree traversal engine

### Platforms Supported
- 📺 YouTube (subscribe + like)
- 🎵 TikTok (follow + like)
- 📸 Instagram Reels (follow + like)

### Notifications
- 📧 Email (beautiful HTML templates)
- 📱 Push Notifications (Firebase FCM)
- 💬 WhatsApp (Twilio API)
- 🤖 Telegram Bot

### Rewards System
- 💰 Points earning and spending
- 💵 Revenue tracking (Stripe payouts)
- 🏆 Leaderboard (daily/weekly/all-time)
- 🎖️ Badges and milestones

### Invite System
- 🔗 Custom vanity URLs (/join/yourname)
- 🎟️ Referral codes (JOHN-ABC1)
- 🔳 QR Code generator
- 📧 Email invite sender

### Apps
- 🌐 Web App (React + Tailwind + Dark Mode)
- 📱 User Mobile App (React Native)
- ⚙️ Admin Mobile App (React Native)
- 🤖 Telegram Bot (aiogram)

### Analytics
- 📈 YouTube Analytics API integration
- 📊 Real subscriber/view tracking
- 📉 Channel growth snapshots
- ✅ Browse success rate tracking

### Multi-Language Support
- 🇺🇸 English
- 🇪🇸 Spanish
- 🇫🇷 French
- 🇸🇦 Arabic
- 🇮🇳 Hindi
- 🇨🇳 Chinese
- 🇧🇷 Portuguese

## 🛠 Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend API | Python FastAPI |
| Task Queue | Celery + Redis |
| Database | PostgreSQL |
| Browser Automation | Playwright |
| Frontend | React + Vite + Tailwind |
| Mobile | React Native + Expo |
| Containers | Docker + Docker Compose |
| Payments | Stripe |
| Push Notifications | Firebase FCM |
| WhatsApp | Twilio |
| Bot | aiogram 3 |
